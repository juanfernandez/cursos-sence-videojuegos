# Curso SENCE - Desarrollo de Videojuegos ONL-19-01-13-0001

Este curso te enseña a identificar las etapas de desarrollo de un videojuego desde su concepción, hasta el proceso de licenciamiento y marketing. Requiere tanto habilidades técnicas en programación, como en comunicación y arte. Aprenderás a utilizar un lenguaje de programación compatible con plataformas móviles y web. También, aprenderás a usar buenas prácticas para el desarrollo y documentación del código, para lograr una correcta colaboración en equipo. Además, aprenderás a actualizar tu computador y captar las tendencias en el desarrollo de videojuegos para mantenerte actualizado.
